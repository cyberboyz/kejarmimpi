# Kejar Mimpi API

Provided API to Kejar Mimpi App

### Installing

run this command:

```
glide install
```
then

```
bee run
```


## Built With

* [Beego](https://beego.me/docs/intro/) - The web frameworks used
* [Glide](https://glide.sh/) - Dependency Management
